# Talebook

Talebook is an open-source collection of agentic UX workflows and AI skills.

The goal is to transform real product design workflows into executable AI systems.

Instead of generating random UI screens, Talebook focuses on:

- UX reasoning
- product thinking
- user flows
- research
- onboarding
- prioritization
- interaction design
- prototype preparation

## Current Focus

- UX Skills Marketplace
- Claude/Codex UX workflows 
- AI-native product design process
- mixing synthetic data with real user input

## Structure

/skills → reusable AI UX skills
/workflows → complete UX flows
/examples → example outputs
/docs → project documentation

Early build in public phase.
